## What's new in /15

Borrowed the-coat-25's zoom-smoothing pattern + added spring-physics zoom_pulse to the controller:

1. **Spring-physics `zoom_pulse`** (in `controllers/taco-kandi.js`) — mass-on-spring with friction. Kicks push velocity, friction returns to rest with smooth elastic easing. No per-frame z-score jitter survives the spring's physical low-pass filter. This is the "animation function" the user asked for.

2. **the-coat-25's smoothstep + cubic-ease pattern** (in shader) —
   - `RAW_INTENSITY` = sum of Normalized inputs (rolling-window smoothed) + spring pulse
   - `smoothstep(0.15, 0.95, ·)` filters out the lowest fluctuations near zero
   - `x³` cubic easing further squashes near-zero, lets peaks pass cleanly
   - Net: the curve is musical instead of twitchy. Same shape an animator uses for a "jab" curve.
