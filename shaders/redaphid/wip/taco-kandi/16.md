# taco-kandi-16

## Origin

Forked from `taco-kandi/15` on **2026-05-09** during the verification harness run while *Death By Love — Walker & Royce* was playing. /16 captures the iter-71 state where the verification harness identified low visible amplitude and we added DIRECT_KICK + tightened the smoothstep upper bound.

## Verification context (carry forward to /16)

Recorded 1104 frames over 55s on 132bpm track. Findings:
- 37 audio kicks, 50 visual events, **81% kick→visual coverage**
- BPM: 132 detected (median kick interval 0.455s) ✓ realistic
- Mean visual lag: −22ms (within 1-2 frame timing noise)
- Mean visual amplitude (dLum): **0.016** — too small per kick
- zoom_pulse: 0 — controller not yet hot-reloaded so the spring is inert (shader still works without it)

## What's new in /16

Same as /15 iter 71:
- **DIRECT_KICK channel** bypasses cubic-ease so strong audio peaks punch the zoom directly. MAX of bassZ + energyZ + fluxZ smoothsteps, full 0..1 range, no ease.
- **Smoothstep tightened** 0.95 → 0.65 upper bound so audio peaks (typically reaching ~0.7 RAW_INTENSITY) saturate cleanly.
- **ZOOM_INTENSITY cap** raised 2.8 → 3.5.
