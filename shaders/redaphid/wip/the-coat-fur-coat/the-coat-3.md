## What Changed from the-coat

This is the same shader code with the session's knob values as the starting point. The vibe: wide shot, thick fur coat as the star, heavy groove sway, subtle eyes/rays (only on drops via the controller sustain).

## Key Decisions

- **knob_2 at -0.346**: The user wanted a pulled-back wide view as the default floor for the zoom range
- **knob_7 at 0.98**: Near-max fur thickness — the coat is chunky and visible
- **knob_12 at 0.669**: Heavy groove — the body sways hard with mids
- **knob_13 at 0**: Fast sustain decay — drops hit and fade quickly, no lingering
- **knob_3/4/5 all low**: Eyes and rays are subtle, letting the coat texture dominate
