## Composition

The branded filibertos taco visualization:
- **Region-tagged taco mask** (shell=red, filling=green, ink=black, outside=transparent in `images/taco.png`)
- **SHELL** (warm tortilla curve) → 8-iter Julia wooli-style fractal with per-device seed uniqueness, reacts to spectralKurtosis (peakedness → c-radius wobble), spectralSkew (tilt → c-coord rotation bias), spectralCrest (spikiness → iter-band contrast), pitchClassMean (slow drift baseline)
- **FILLING** (lettuce/textured top) → plasma raymarch + AURORA RIBBONS (3 independent feature-domain bands)
- **INK** → legitimately black with subtle warm sheen — logo discernable through any chaos
- **OUTLINE** → photon ring (knob_12 + audio-pumped radius via bass_smooth + drop_glow), OUTLINE_RADIATION concentric waves, edge glow halo, prismatic edge split
- **VOID** → starfield exterior

## Audio reactivity

- Trend-dominant zoom-pulse (R²-weighted energySlope + bassSlope confident builds, single bounded INSTANT accent on kicks). No `beat` uniform — z-score multi-signal kick detector instead.
- knob_12 outline photon ring radius pumps with bass_smooth + drop_glow, sharpness breathes (tighter on calm, softer on heavy bass).
- Shell fractal breathes with multiple independent musical textures.
- All flash-prone effects banding-safe via bounded mix() blends.

## Palette (locked, magenta impossible)

- CORE_HUE: 0.55-0.75 rad (orange family) ± 0.05 rad seed tilt
- CORONA_HUE: 4.0-4.4 rad (deep blue-violet) ± 0.05 rad seed tilt
- All hue rotations bounded — magenta (3.0-3.5 rad) is unreachable via any path

## Per-device uniqueness (kandi bracelets)

- `seed`  → wooli-fractal Julia c-radius family, outline-tunnel ring count
- `seed2` → CORE_HUE warm tilt, shell-fractal c-coord phase, warm hearth
- `seed3` → CORONA_HUE photon-ring tilt
- `seed4` → CORONA_HUE outer tilt
- All seeded contributions bounded so brand stays in family
