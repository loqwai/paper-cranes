## History of changes
- **Iter 18 (2026-05-09) — *I Don't Love You / MCR* — KALEIDOSCOPE SEAM FIX** — User flagged "horizontal seam." Root cause: `atan(p.y, p.x)` returns angle in `[-π, +π]` with the discontinuity falling along the negative-x axis (left horizontal). My `mod(kal_a + iTime*0.05, seg)` then `abs(kal_a - seg*0.5)` was preserving that discontinuity. **Fix:** add `+PI` first to map angle into `[0, TAU]` (continuous), do the mod, then subtract `PI` when reconstructing `vec2(cos, sin) * r`. Also force `kaleido_n` to be even via `floor(N*0.5)*2` so mirror boundaries align (odd folds create a second visible discontinuity).
- **Image swap (2026-05-09)** — replaced `images/taco.png` with `~/Projects/filibertos/taco.png`. New taco is larger, centered, with bolder ink lines. Converted near-white pixels to alpha=0 so the mask logic (`texel.a * (1.0 - texel.r)`) reads it correctly.
