## Key lessons carried forward (from the-coat-15)
- **Slow modulations vs fast modulations** read as opposite emotional states. Sub-1Hz = brooding monster. 5Hz+ = jittery sparkly. Match modulation frequency to emotional intent.
- **Match modulation frequency to emotional intent** — slow modulations on a single body should breathe as one organism (share a base frequency).
- **Plasma is the opposite of the-coat aesthetic**: no body/figure, no narrative anchor — it's pure spectral fluid. Push aesthetic toward TIMBRAL response (treble shimmer, entropy chaos, centroid hue) over BASS-WEIGHTED monster moves.
- **Avoid stacking >6 simultaneous overlay effects** — composition blows out. Plasma starts with just one effect (the plasma itself), so we have headroom.
- **Use the #define swap pattern** (commented constant alternates) so each parameter can be auditioned with `?noaudio=true`.
- **Statistical variations matter**: prefer `*ZScore` for events, `*Normalized` for smooth modulation, `*Mean`/`*Median` for slow drift.
- **`*Mean` for energy is tiny (~0.003-0.01)** — don't multiply by it expecting a real modulation. Use `Normalized` for visible motion.
- **Knob auto-wire**: when user grabs an unwired knob and twists it >0.02, that's a gesture to map it to something interesting that tick.
